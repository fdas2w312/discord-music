💡 Как это работает

  1. Слова набираются по одному (или по N штук за тик)
  2. Когда предложение заканчивается (точка, !, ?, перенос строки) — оно показывается целиком
  3. Затем статус очищается и начинается следующее предложение
  4. После последнего предложения — начинается сначала
